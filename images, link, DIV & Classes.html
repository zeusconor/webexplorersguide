Mastering Images, Links, and Layouts

1. The Image Element (<img>)

The <img> tag is used to embed images. It is an empty element (no closing tag).

Basic Syntax

  • src: URL or path to the image file.
  • alt: Alternative text (Shows if the image fails to load).
Copy
<img src="nature.jpg" alt="A green forest mountain" width="300">

💡 Pro Tips for Images

Tip 1: Always include Alt text

This is crucial for SEO and accessibility for visually impaired users using screen readers.

Tip 2: Make it Responsive

To prevent images from overflowing on mobile screens, always use CSS like max-width: 100%; height: auto;.

Tip 3: Image as a Link

You can make an image clickable by wrapping it inside an anchor <a> tag.

Live Example: Image as a Link

Click the image below to visit Unsplash:

<a href="https://unsplash.com" target="_blank">
  <img src="mountain.jpg" alt="Mountain Lake">
</a>

2. The Link Element (<a>)

The <a> (Anchor) tag is like a "portal" that takes you to another place on the internet.

href (Hyperlink Reference)

The destination URL. Where does the link go?

target="_blank"

Opens the link in a new tab so users don't leave your site.

🚀 Pro Tips for Links

✔

Descriptive Text: Don't just write "Click Here". Write "Download the PDF" or "Read more about cats". It helps Google understand your page better!

✔

Internal vs External: Use target="_blank" for other websites, but don't use it for your own pages.

✔

Title Attribute: Add title="Visit Google" to show a small tooltip when someone hovers over the link.

Copy
<!-- Opening in a new tab with a title -->
<a href="https://google.com" target="_blank" title="Search on Google">
    Search Something!
</a>
Search Something!

3. Div & Classes

Think of a <div> as an invisible Cardboard Box. It organizes your stuff!

What are Classes?

A class is like a label you put on your boxes. If you have 10 boxes labeled "blue-box", you can tell CSS to make all "blue-box" items have a blue background at once!

🔥 Power User Pro Tips

  • 1. Meaningful Names: Name classes based on what they ARE, not what they look like. Use .news-card instead of .big-red-box.
  • 2. Reusability: The superpower of a class is that you can use it on many elements. One style change updates everything!
  • 3. Multiple Classes: An element can have more than one class. Just separate them with a space:
    <div class="card shadow rounded">
Copy
<!-- One container div with two children inside -->
<div class="parent-container">
    <div class="child-box highlighted">Box A</div>
    <div class="child-box">Box B</div>
</div>
